# SERVER

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
pull pfes-server vào thư mục /home/pfes/pfes-server
npm run build
pm2 start app.js --name server
pm2 startup && pm2 save

```
